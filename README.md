# Awesome font

Custom font to mimic Awesome's workspace tag symbols in polybar.

### Supported characters

| Code      | Character        |
| --------- | ---------------- |
| U+0032    | space            |
| U+E000    | squares sel fill |
| U+E001    | squares sel      |
| U+E002    | squares unsel    |
